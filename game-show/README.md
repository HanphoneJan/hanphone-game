# GameHub - 现代游戏合集

一个包含30个HTML5游戏的现代化网站，具有酷炫的设计和流畅的用户体验。

## 🎮 项目特色

- **30个精选游戏** - 包含各种类型的HTML5游戏
- **现代化设计** - 采用未来感的设计风格，渐变色彩和流畅动画
- **响应式布局** - 完美适配桌面和移动设备
- **动态背景** - 使用Vanta.js创建交互式云层背景
- **平滑动画** - 滚动动画、悬停效果和过渡效果

## 🎨 设计特色

### 视觉效果
- **渐变色彩** - 使用蓝紫色渐变创造科技感
- **动态背景** - Vanta.js云层动画背景
- **霓虹发光** - 按钮和标题的发光效果
- **流畅动画** - CSS3动画和过渡效果

### 交互体验
- **悬停效果** - 卡片和按钮的3D悬停效果
- **滚动动画** - 元素进入视口时的淡入动画
- **响应式设计** - 适配各种屏幕尺寸
- **平滑滚动** - 页面内导航的平滑滚动

## 🛠 技术栈

### 前端技术
- **HTML5** - 语义化标记
- **CSS3** - 现代样式和动画
- **JavaScript ES6+** - 交互逻辑
- **Tailwind CSS** - 响应式设计

### 第三方库
- **Vanta.js** - 动态背景效果
- **Three.js** - 3D图形渲染
- **Font Awesome** - 图标库
- **Google Fonts** - 字体库

## 📱 响应式设计

网站采用移动优先的设计方法，确保在各种设备上都有出色的表现：

- **桌面端** (1200px+) - 多列网格布局
- **平板端** (768px-1199px) - 两列网格布局
- **移动端** (< 768px) - 单列布局，优化触控体验
